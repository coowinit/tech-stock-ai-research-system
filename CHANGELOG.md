# 更新日志

## v1.2.1 数据管理统一版

- 删除 `index.html` 中的数据库导入、导出、同步和恢复初始数据按钮，使核心看板只负责查看、筛选、分析和打印。
- 将所有数据库维护操作集中到 `screening.html` 的“候选池与数据管理”标签。
- 新增候选池工作流 JSON 导入功能，并保留工作流导出。
- 删除“导出全部快照”，本地快照明确定位为当前浏览器中的回退记录。
- 本地快照最多保留 20 个，支持手动创建、刷新、恢复和删除。
- 导入正式数据库、同步内置数据、正式核验入库、恢复快照和恢复初始数据前，自动保留操作前快照。
- 统一核心看板中的运行提示，引导用户到筛选中心完成数据管理。

## v1.1.1 数据同步修复

- 修复浏览器旧 `localStorage` 数据长期覆盖新版内置数据的问题。
- 新增 `builtInDataVersion` 内置数据版本标记。
- 页面加载时自动把新版 2025A、2026Q1 数据合并到旧数据库。
- 自动同步时保留用户新增公司、额外报告期、备注和观察状态。
- 新增“同步最新版内置数据”按钮，可手动重新同步而不清空扩展数据。
- “恢复初始数据”仍作为完全重置功能，会清除自行新增的数据。
