## 数据处理原则

1. 7家公司全部纳入正式数据库，以保证物理AI产业链完整；
2. 继续保留 `exclude688: true`，默认看板仍排除科创板；
3. 产业龙头身份与财务高增长结果分开显示；
4. 每个报告期均记录营业收入、归母净利润、扣非净利润、经营现金流和正式公告来源；
5. 核验状态统一标记为 `matched`，并保留字段级证据文本。

## 重点风险

- 半导体材料、晶圆制造与AI芯片属于物理AI底层基础设施，不等同于机器人业务直接收入；
- 高增速可能受到低基数、交付节奏或行业周期影响；
- 中无人机2025年经营现金流为负，订单与回款节奏需要结合观察；
- 奥比中光2026Q1经营现金流转负，需继续关注盈利质量；
- 科创板公司波动和估值风险通常更高，系统默认排除设置保持不变。
